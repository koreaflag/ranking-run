■ 미완성 / 작업 필요한 부분

[프론트]
- 리뷰 UI (StarRating, ReviewSection) 스텁 상태
- 마이페이지 서브화면 (설정, 내코스, 프로필편집) 스텁
- 월드 화면 날씨/이벤트 미완
- 홈화면 위치 하드코딩 (서울 좌표) → 실제 GPS 필요
- GPS 청크 백그라운드 업로드 로직 미구현

[백엔드]
- 테스트코드 (pytest) 미작성
- Rate Limiting 미적용
- 감사 로깅 미적용
- 파일스토리지 로컬 → S3 전환 필요
- CORS 프로덕션 도메인 미설정

[프로덕션 필수]
- JWT_SECRET_KEY 강력한 키로 교체
- 카카오/Apple OAuth 크레덴셜 발급
- dev-login 엔드포인트 비활성화
