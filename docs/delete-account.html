계정 삭제 방법

  1. 앱 내에서 삭제: RUNVS 앱 → 마이페이지 → 설정(톱니바퀴) → 계정 삭제
  2. 이메일로 요청: 아래 이메일로 가입 시 사용한 이메일 주소와 함께 삭제 요청을 보내주세요.
