RUNVS Location Data Collection & Use Consent

RUNVS Service (FLAG)

Effective Date: March 22, 2026

This consent form explains how RUNVS collects, uses, and processes your location data in accordance with applicable laws, including the Korean Location Information Act.

1. Location Data Collected

  • Real-time GPS coordinates (latitude/longitude) via your smartphone
  • Collection method: Device location services (iOS Core Location / Android Location API)

2. Purpose of Location Data Use

Purpose Details
Running Recording Record running route, distance, pace, and elevation
Course Creation Create shareable courses from your running routes
Rankings Compare records and calculate course rankings
Course Navigation Provide route guidance when running saved courses
Service Improvement Analyze GPS accuracy and improve route quality

3. Retention Period

  • Running route data: Retained until account deletion or user deletion request
  • Location collection/use confirmation records: Retained for 6 months then destroyed

4. Right to Refuse

You may refuse to consent to location data collection. However, if you refuse, running recording, course creation, and ranking features will be unavailable.

5. Collection Notification

When you start a running session, the app displays an active indicator to notify you that location data is being collected. Collection stops immediately when the session ends.
