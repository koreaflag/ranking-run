RUNVS 位置情報の収集・利用に関する同意書

施行日: 2026年3月22日

RUNVSサービス (FLAG)

本同意書は、RUNVSが位置情報を収集・利用する方法について説明するものです。

1. 位置情報の収集項目

  • スマートフォンGPSによるリアルタイム位置座標（緯度・経度）
  • 収集方式: ユーザーデバイスの位置サービス（iOS Core Location / Android Location API）

2. 位置情報の利用目的

利用目的詳細内容
ランニング記録ランニング経路、距離、ペース、高度の記録・保存
コース作成ユーザーが走った経路をコースとして作成・共有
ランキング算出コース別記録比較及び順位算出
コースナビゲーション保存されたコースに沿って走る際の経路案内
サービス改善GPS精度分析及び経路品質改善

3. 位置情報の保有期間

  • ランニング経路データ: 退会またはユーザーの削除要請時まで
  • 位置情報収集・利用事実確認資料: 6ヶ月保管後破棄

4. 同意拒否の権利及び不利益

ユーザーは位置情報の収集・利用に対する同意を拒否することができます。ただし、同意拒否時はランニング記録、コース作成、ランキング機能の利用ができません。

5. 収集の通知

ユーザーがランニングセッションを開始すると、アプリ内表示により位置情報収集中であることを通知します。セッション終了時に収集が即時停止されます。
